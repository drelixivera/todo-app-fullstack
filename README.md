📝 Full-Stack Todo Orchestrator
A production-ready Task Management application built to demonstrate mastery of the MERN stack (MongoDB, Express, React, Node.js). This projects showcase my ability to architect both frontend and backend logic within a single, organized monorepo.

🛠️ Technical Stack
    Frontend: React.js powered by vite for optimized build speeds.
    Backend: Node.js & Express.js for robust API routing.
    Database: MongoDB for flexible, document-based data storage.
    Styling: Responsive CSS designed for a seamless mobile and desktop experience.

✨ Core Functionalities
    Full CRUD Lifestyle: Users can create, view, edit and delete tasks with instant UI updates
    Integrated API: A custom-built REST API that handles all task data persistence.
    Optimized Performance: Fast load times and Hot module replacement (HMR) via vite.
    Environment Safety: Configured for secure environment variable management (e.g .env.example).

📂 Architecture
    This project is structured as a monorepo to maintain a tight link between the client and server: 
    /TODO-APP-FULLSTACK
    ├── /todo-app-backend  # Server-side logic & API
    └── /Todo-app-frontend # Client-side React application

⚙️ How to Run Locally
1. Clone the Repo: git clone [https://github.com/Drelix-Ivera/todo-app-fullstack.git](https://github.com/Drelix-Ivera/todo-app-fullstack.git)
2. Server Setup: Navigate to /todo-app-backend, run npm install, and configure your .env
3. Client Setup: Navigate to /Todo-app-frontend and run npm install.
4. Launch: Start the backend with npm start and the frontend with npm run dev.
